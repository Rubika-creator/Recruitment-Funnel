# Recruitment-Funnel Analysis
**Objective**
Analyze candidate progression through hiring stages and identify drop-off points.

**Dataset**
Synthetic dataset with stages:
Applied
Screening
Interview
Offer
Hired

**Tools Used**
SQL, Power BI

**Analysis Performed**
Stage-wise candidate counts
Drop-off rate between stages
Time-to-hire
Average time-to-hire
Conversion rates between stages

**Key Insight**
Highest drop-off occurs at Offer → Hired (~48.6%)
Indicates potential issues in offer acceptance

**SQL Concepts Used**
CTE (Common Table Expressions)
CASE statements
Self Join
Aggregations
**Power BI Concepts Used**
 DAX
 Funnel Chart
 Table
 Cards for Key Metrics
